# Included

- public incident submission
- optional evidence upload
- tracking by report ID + token
- authenticated admin dashboard
- status/severity/category management
- audit-style fields and logs
- secure backend validation
- low-cost observability

# Excluded from MVP

- full public user accounts
- email notification system
- malware scanning pipeline
- advanced analytics dashboards
- custom domain
- IaC-first deployment
- multi-environment release strategy
